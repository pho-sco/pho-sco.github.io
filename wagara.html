Description

A variational autoencoder (VAE) and a generative adverserial network (GAN) are trained with repeating image patterns. The autoencoder has a latent space of 10 where each parameter is forced to a normal distribution. The ability to match input and output images was optimized during training. Additionally, a loss on the image edges was defined to improve the quality of the output when placing images next to each other. Inference of the decoder returns an image of 96 x 96 pixels from an input vector of 10 elements. 10 x 10 images are predicted where two parameters of the input are varied linearly along the x- and y-axis.

The generator of the GAN uses an input of 100 random values to generate patterns. The image output is created in the same way as for the VAE. Please consider the complexity of the different models when predicting images. The GAN's generator is larger in size and therefore requires more time for a prediction. The GAN is able to generate images with sharp edges, resulting in fabrique-like patterns. The VAE on the other hand yields more roundish images.
